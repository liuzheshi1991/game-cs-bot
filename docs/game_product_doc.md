# 游戏产品文档（客服机器人知识库原始版）

## 1. 产品概述

- **产品名称**：幻域远征（示例名，可替换）
- **产品类型**：移动端多人在线角色扮演游戏（MMORPG）
- **平台**：iOS / Android
- **目标用户**：18-35 岁，偏好社交、公会协作、成长养成玩法的玩家
- **客服机器人定位**：7x24 小时处理高频咨询，覆盖账号、充值、活动、玩法、处罚申诉等问题

## 2. 核心玩法说明

### 2.1 职业体系

- 战士：近战、高生存、控制能力强
- 法师：远程、高爆发、范围输出
- 游侠：远程、持续输出、机动性高
- 牧师：治疗、增益辅助、团队保障

### 2.2 成长系统

- **等级系统**：主线任务、日常任务、副本经验获得
- **装备系统**：普通/稀有/史诗/传说品质，强化与镶嵌
- **宠物系统**：宠物培养、技能学习、战斗助阵
- **天赋系统**：多分支加点，支持重置

### 2.3 社交系统

- 公会创建与加入
- 组队副本与语音协作
- 世界频道、公会频道、私聊系统

### 2.4 赛季与活动

- 赛季时长：约 8-10 周
- 活动类型：签到活动、限时副本、冲榜活动、节日活动
- 奖励发放：道具邮件、称号、外观、钻石返利

## 3. 账号与登录机制

- 支持手机号验证码登录、游客登录、第三方平台登录
- 游客账号支持绑定手机号
- 异常登录触发二次验证
- 支持跨端登录（同账号 iOS/Android 数据互通，具体按渠道政策）

## 4. 付费与订单体系

### 4.1 货币说明

- 金币：游戏内通用货币
- 钻石：充值货币，可用于抽卡、商城购买
- 绑定钻石：活动赠送，部分场景可用

### 4.2 充值规则

- 充值到账通常为实时到账，偶发延迟 1-10 分钟
- 未到账可通过订单号发起核单
- 苹果支付受 App Store 回调影响，可能存在延时

### 4.3 退款规则（示例）

- 未成年人误充值：按平台规范提供证明材料审核
- 一般情况下，已消耗虚拟物品订单不支持退款
- 退款成功后可能触发道具回收或账号封禁（防止套利）

## 5. 安全与处罚体系

- **违规类型**：外挂、脚本、恶意刷资源、辱骂、诈骗、代充
- **处罚等级**：警告、禁言、短期封禁、永久封禁
- **申诉流程**：提交 UID + 时间 + 说明 + 截图，客服 1-3 个工作日处理

## 6. 常见异常场景定义

- 登录失败（网络、版本过低、服务器维护）
- 充值未到账（支付成功但钻石未增加）
- 道具丢失（误分解、网络异常导致显示问题）
- 活动奖励未收到（延迟发放、条件未满足）
- 匹配异常（高峰期排队时间过长）

## 7. 客服机器人业务目标

- 高频问题自动解决率 >= 70%
- 转人工率降低 30%
- 首次响应时间 <= 3 秒
- 用户满意度（CSAT）>= 85%

## 8. 客服知识库标签体系（用于 RAG 检索）

- 一级标签：账号、登录、充值、活动、玩法、社交、公会、处罚、申诉、技术故障
- 二级标签示例：
  - 账号-绑定/找回/异常登录
  - 充值-未到账/重复扣费/退款
  - 活动-规则说明/奖励发放/资格条件
  - 处罚-禁言/封号/申诉进度
- 情绪标签：紧急、投诉、咨询、建议

## 9. 客服响应规范（机器人）

- 先安抚后解决：先确认用户问题，再给步骤
- 避免绝对化表达：使用“通常/一般/建议”
- 涉及账号安全时，必须引导至身份验证流程
- 涉及退款与处罚时，明确“需人工审核”

- 每周更新 FAQ 与活动规则
- 每次大版本上线前完成知识库重建与回归测试
